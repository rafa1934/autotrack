# AutoTrack

Sistema de gerenciamento de Ordens de Serviço para oficinas mecânicas.

## Funcionalidades

- Login por perfil (Oficina e Gestor)
- Cadastro de Ordens de Serviço
- Consulta de Ordens de Serviço
- Alteração de valores da Ordem de Serviço
- Aprovação de Ordem de Serviço
- Reprovação de Ordem de Serviço
- Registro de observações do gestor
- Histórico das Ordens de Serviço

## Tecnologias Utilizadas

- PHP
- MySQL
- HTML
- CSS
- JavaScript

## Como Executar

1. Instalar o XAMPP.
2. Copiar a pasta AutoTrack para:

C:\xampp\htdocs

3. Iniciar Apache e MySQL.
4. Importar o banco de dados localizado na pasta:

sql/

5. Abrir o navegador:

(https://autotrackrafa.infinityfreeapp.com/AutoTrack/)
